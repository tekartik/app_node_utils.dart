# Setup

## Pure node project

Dependencies:
```yaml
dependencies:
  tekartik_app_node_utils:
    git:
      url: https://github.com/tekartik/app_node_utils.dart
      path: node_utils
      ref: dart2_3
    version: '>=0.1.0'
```
## Usage

A simple usage example:

```dart
import 'package:node_utils/node_utils.dart';

main() {
  var awesome = new Awesome();
}
```
